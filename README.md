# Bilisound 移动客户端

基于 React Native (Expo) 的 Bilisound 移动客户端。

## 开发模式

由于本项目引用了若干第三方库，因此必须使用 [Development Build](https://docs.expo.dev/guides/local-app-development/)。**不支持 Expo Go。**

### iOS

```bash
npm run ios
```

### Android

```bash
npm run android
```

## 构建生产版本

构建生产版本以前，需要准备好 keystore 文件 (Android)……

### Android

```bash
npm run build:android
```
